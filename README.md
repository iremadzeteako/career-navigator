# 🧭 Career Navigator

პროფესიული ორიენტაციის ვებ-აპლიკაცია — HTML5 + Tailwind CSS + Vanilla JavaScript

---

## 📁 პროექტის სტრუქტურა

```
career-navigator/
├── index.html        ← მთავარი HTML გვერდი (სტრუქტურა)
├── css/
│   └── style.css     ← ყველა სტილი (CSS ცვლადები, Glass, ანიმაციები)
├── js/
│   └── app.js        ← ყველა ლოგიკა (მონაცემთა ბაზა + ალგორითმი)
└── README.md         ← ეს ფაილი
```

---

## 🚀 გაშვება

1. ჩამოტვირთე ZIP და გახსენი საქაღალდე
2. გახსენი `index.html` ნებისმიერ ბრაუზერში (Chrome / Firefox / Edge)
3. ინტერნეტ კავშირი საჭიროა: Tailwind CDN + Google Fonts + YouTube ვიდეოები

> ⚠️ **შენიშვნა:** ლოკალური ფაილიდან გახსნისას (`file://`) YouTube iframe შეიძლება დაბლოკილი იყოს ზოგიერთ ბრაუზერში. ამ შემთხვევაში გამოიყენე ლოკალური სერვერი (VS Code Live Server, Python `http.server` და ა.შ.).

## 🧠 მატჩინგ ალგორითმი

```
score = (matched_required / total_required) × 80
      + (matched_optional / total_optional) × 20
```

- **Required უნარები** — 80% ქულა
- **Optional უნარები** — ბონუს 20% ქულა
- TOP-3 პროფესია სორტდება კლებადობით
